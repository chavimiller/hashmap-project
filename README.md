# hashmap-project

The class includes methods for adding, retrieving, updating, and removing entries, as well as utilities for accessing all keys, values, and entries.

The project focuses on understanding hashing functions, modular design, and algorithmic problem-solving.
The hashmap is then tested in a separate file.
